## ‚úçÔ∏è Usage Guide

* **Expression Input**: Use the input box or buttons to enter expressions.
* **Evaluate**: Click `=` or press `Enter` to compute the result.
* **Clear**: Click `AC` to reset both the result and expression.
* **Backspace**: Use `‚Üê` button or Backspace key to remove last character.
* **Keyboard Shortcuts**:

  * Numbers: `0-9`
  * Operators: `+`, `-`, `*`, `/`, `x^y`
  * Decimal: `.`
  * Parentheses: `(`, `)`
  * Evaluate: `Enter`
  * Clear: `C` or `Escape`
